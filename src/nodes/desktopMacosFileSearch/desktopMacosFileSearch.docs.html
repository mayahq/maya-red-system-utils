Description

Search for files and apps on MacOS.

Config

query str
String input to search for file and apps on MacOS

Usage

It takes the following inputs:

query: name of the file/search term
kind: The file type (default is 'app'). If msg.payload.kind is selected, it selects the value from msg.payload.kind
Directory: Directory inside which you want to search for the file

If the node is pre-configured by the user, it takes that value, otherwise it fetches values from msg.payload
