Description

Send custom native notifications on MacOS.


Config

titlestring
Title heading of the desktop notification sent.
messagestring
Main message of the desktop notification.
closeBtnstring
Label to put on the close notification button.
actionBtnstring
Label to put on the custom action button on the notification.
replycheckbox
Get reply from user on showing notification?
soundcheckbox
Enable default notification sound?
waitcheckbox
Wait for user action before timeout in 5s?

Outputs

  1. onClick
    If notification is just clicked, then msg is outputted from this port. Metadata in msg.notification.
  2. onReply
    If user replies to the notification, then msg is outputted from this port, find reply at msg.notification.reply. Make sure reply is switched to true.
  3. onAction
    If user clicks on the actionBtn, then msg is outputted from this port.Metadata in msg.notification.
  4. onTimeout
    If notification is not interacted with, then msg is outputted from this port after timeout of 5 seconds. Metadata in msg.notification.
  5. onClose
    If notification is closed manually from the closeBtn, then msg is outputted from this port. Metadata in msg.notification.
