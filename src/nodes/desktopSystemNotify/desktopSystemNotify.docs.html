Description

Send cross platform native notifications on desktop. Standard usage, with cross-platform fallbacks.


Config

title string
Title heading of the desktop notification sent.
message string
Main message of the desktop notification.
sound checkbox
Enable default notification sound?
wait checkbox
Wait for user action before timeout in 5s?

Outputs

  1. onClick
    If notification is clicked, then msg is outputted from this port.
  2. onTimeout
    If notification is not clicked, then msg is outputted from this port after timeout of 5 seconds.
